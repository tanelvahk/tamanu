Smoothly scroll to top of the page.Change your own back to top image in admin section.

== Description ==
Now a Days websites have infinite scroll.Use this plugin user just click on the bottom icon it will smoothly go back to top and also easily change the back to top icon in admin section.

= Features Included =
* Simple to use just install and visit your site and scroll down automatically appear back to top icon.Just click on it, goes to top smoothly.
* Change your own back to top image in admin section.


= Usage =
* Once install the plugin go to settings menu -> Back To Top-SBTT
* Default Back To Top image appear, If you want to change, you can change it.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once install the plugin go to settings menu -> Back To Top-SBTT
4. Default Back To Top image apper, If you want to change, you can change it.
